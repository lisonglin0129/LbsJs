## 开始使用
### 引入文件：
```html
	<script src="js/lbs.min.js"></script>
```	
* 通过Lbs.get.Id来获取html元素dom输

	<head>
		<meta charset="utf-8">
		<title>libJs</title>
		<script src="js/lbs.min.js"></script>
	</head>
	<body>
	  <div id=""></div>
	</body>
</html>
